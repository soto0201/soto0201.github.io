ソフ演3を履修中止しました

理由として

なんかいろいろあって速攻で更新が止まってしまっていました。猛省してます。
理由としては少しメンタルも体調もやってしまっていたためです。

最近は全国で緊急事態宣言が解除されるなどして落ち着いて来ましたが、世間はリモートで授業や仕事などに取り組むという風潮です。
自分は普段から在宅でバイトをするなどして慣れているのですが、親は機械音痴のため最低限仕事ができるように面倒をみていました。
この時点でなぜか親の仕事がかなり増えていたため、結果として自分が休む時間が無くなるという状況になっていました。

この時点で考え直すべきなのですが、休む暇が無いのにさらに授業の準備に長時間必要になると睡眠時間を削るしかなく、
その時は「意地でもやってやるぞ！」と思っていたのもあり、無理をしてしまい寝込む形になってしまいました…
やはり無理は良くない…ということで今切り捨てられる物の中で一番コストの重いソフ演を履修中止してきました。

履修中止は正直死ぬほど悔しいですし、メンバーの人(もうすでにあと一人しかいない)にも申し訳なさがありますが、
こうしないと本当に自分が過労死するのではないのかと思っていたのでやむなし…という感じです。

寝込んでいた時に

メンタルを崩した当初は指を動かすのもしんどいという状況だったので、Amazon primeでアニメを自動再生してそれを観ていました。
なろう系のアニメをみていたのですが、なろうは結構愛読していて忌避感もないので普通に楽しんでメンタルを回復させていました。
選んだタイトルが良かったのかもしれませんが…

ある程度回復してきたらシャニマスをやるなどしていました。
なんとなくやっていたのですが、自力で霧子ちゃんの True End に辿り着いたときにメンタルはだいぶ本調子に戻っていました。
やはり自分は落ち込んだときにゲームをするのが一番手っ取り早いのかもしれないです。
そういうわけでシャニマスには感謝の意を表し、iTunes Storeで既存のシャニマスの曲を全て購入してきました。
今は作業中などに聞いたりしています。Dye the sky. はとても良い曲です。聴きましょう。ここから聴けます

今後の予定としては

最近は休む間もなくずっと作業をしてしまったためにメンタルも体調も崩したので、定期的に休みを設けることにしています。
最低でも1週間に1日くらいは自由にできる日を設けようかな…という所存です。
まあ、その時に親の仕事を手伝っているので休んでないのでは？と思っているのですが…

あとは原因として、もしかしたらストレスが溜まっていたのかもしれません…
自粛で大学に入れないのでingressはできないですし、趣味の外食ができなかったのが一番影響を与えていたのかもしれません…
今後は余裕を持って行動をしていくよう心掛けるようにします。

余談

やはり記事を書いていると見栄えのために画像が欲しくなりますね。今回だと霧子のTrue Endの画像とか貼れたら最高だと思います。
そのためには例のTwitterアカウントの鍵を外さないといけない…
ぐぬぬ…考えておきましょう…
